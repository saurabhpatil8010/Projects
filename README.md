Delivery Management System

Technologies Used :- Python,Flask(PythonWebFramework),HTML,CSS,JavaScript,Bootstrap

Delivery Management System is a web-based application developed using Python (Flask) 
and MySQL that helps manage product deliveries efficiently. The system allows users to 
manage orders, delivery details, customer information, and track delivery status.

The application includes a responsive user interface built using HTML, CSS, JavaScript, and 
Bootstrap. The backend handles business logic, user authentication, and database 
operations, while MySQL stores order and customer data securely.

The system improves delivery tracking, reduces manual work, and ensures organized order 
management.
